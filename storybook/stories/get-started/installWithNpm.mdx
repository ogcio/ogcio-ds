<h1>Install with Node.js package manager (npm)</h1>
<h2>Requirements</h2>
<ol>
  <li>
    <p>
      <a href="https://nodejs.org/en/" target="_blank" rel="noopener">
        Install Node.js
      </a>
      .
    </p>
  </li>
  <li>
    <p>
      <code>cd</code> to the root of your project and check if you have a{' '}
      <a
        href="https://docs.npmjs.com/files/package.json"
        target="_blank"
        rel="noopener"
      >
        <code>package.json</code> file
      </a>
      . If you do not have the file, create it by running:
    </p>
    <CodeSnippet code="npm init" />
  </li>
  <li>
    <p>Install the OGCIO-DS package</p>
    <CodeSnippet code="npm install @ogcio/ogcio-ds --save" />
    <p>
      When the installation finishes, the <code>@ogcio/ogcio-ds</code> package
      will be in your <code>node_modules</code> folder.
    </p>
    <p>
      You should now get started by [getting the CSS, assets and JavaScript
      working](/?path=/docs/docs-get-started--page) with one gov.ie Frontend
      component.
    </p>
  </li>
</ol>

<h1>Sass API Reference</h1>

<h2>Settings</h2>

<h3>Assets</h3>

<h4>$govie-assets-path</h4>
<p>Path to the assets directory, with trailing slash.</p>
<p>
  This is the directory where the images and fonts subdirectories live. You will
  need to make this directory available via your application - see the README
  for details.
</p>
<h5>Default value</h5>
<Code code={`$govie-assets-path: "/assets/";`} />

<h4>$govie-images-path</h4>
<p>Path to the images folder, with trailing slash.</p>
<h5>Default value</h5>
<Code code={`$govie-images-path: "#{$govie-assets-path}images/";`} />

<h4>$govie-fonts-path</h4>
<p>Path to the fonts folder, with trailing slash.</p>
<h5>Default value</h5>
<Code code={`$govie-fonts-path: "#{$govie-assets-path}fonts/";`} />

<h3>Colours</h3>
<h4>$govie-brand-colour</h4>
<p>Brand colour.</p>
<h5>Default value</h5>
<Code code={`$govie-brand-colour: govie-colour("blue");`} />

<h4>$govie-text-colour</h4>
<p>Text colour.</p>
<h5>Default value</h5>
<Code code={`$govie-text-colour: govie-colour("black");`} />

<h4>$govie-body-background-colour</h4>
<p>Body background colour.</p>
<h5>Default value</h5>
<Code code={`$govie-body-background-colour: govie-colour("white");`} />

<h4>$govie-print-text-colour</h4>
<p>Text colour for print media.</p>
<p>Use 'true black' to avoid printers using colour ink to print body text.</p>
<h5>Default value</h5>
<Code code={`$govie-print-text-colour: #000000;`} />

<h4>$govie-secondary-text-colour</h4>
<p>Secondary text colour.</p>
<p>Used in for example 'muted' text and help text.</p>
<h5>Default value</h5>
<Code code={`$govie-secondary-text-colour: govie-colour("dark-grey");`} />

<h4>$govie-focus-colour</h4>
<p>Focus colour.</p>
<p>
  Used for outline (and background, where appropriate) when interactive elements
  (links, form controls) have keyboard focus.
</p>
<h5>Default value</h5>
<Code code={`$govie-focus-colour: govie-colour("yellow");`} />

<h4>$govie-focus-text-colour</h4>
<p>Focused text colour.</p>
<p>
  Ensure that the contrast between the text and background colour passes WCAG
  Level AA contrast requirements.
</p>
<h5>Default value</h5>
<Code code={`$govie-focus-text-colour: govie-colour("black");`} />

<h4>$govie-focus-text-colour</h4>
<p>Focused text colour.</p>
<p>
  Ensure that the contrast between the text and background colour passes WCAG

<Code code={`$govie-focus-text-colour: govie-colour("black");`} />

<h3>Typography</h3>

<h4>$govie-font-family-lato</h4>
<p>
  List of font families to use if using Lato (the default font 'stack for
  OGCIO-DS)
</p>
<h5>Default value</h5>
<Code code={`$govie-font-family-lato: 'Lato', arial, sans-serif;`} />

<h4>$govie-font-family</h4>
<p>Font families to use for all typography on screen media.</p>
<h5>Default value</h5>
<Code code={`$govie-font-family: $govie-font-family-lato;`} />

<h4>$govie-font-family-print</h4>
<p>Font families to use for print media.</p>
<p>
  We recommend that you use system fonts when printing. This will avoid issues
  with some printer drivers and operating systems.
</p>
<h5>Default value</h5>
<Code code={`$govie-font-family-print: sans-serif;`} />

<h4>$govie-include-default-font-face</h4>
<p>Include the default @font-face declarations.</p>
<p>
  If you have set <code>$govie-font-family</code> to something other than{' '}
  <code>$govie-font-family-lato</code> this option is disabled by default.
</p>
<h5>Default value</h5>
<Code
  code={`
$govie-include-default-font-face: (
  $govie-font-family == $govie-font-family-lato
);`}
/>

<h4>$govie-font-weight-govie</h4>
<p>Font weight for govie typography</p>
<h5>Default value</h5>
<Code code={`$govie-font-weight-govie: 400;`} />

<h4>$govie-font-weight-bold</h4>
<p>Font weight for bold typography</p>
<h5>Default value</h5>
<Code code={`$govie-font-weight-bold: 700;`} />

<h4>$govie-root-font-size</h4>
<p>Root font size.</p>
<p>
  This is used to calculate rem sizes for the typography, and should match the
  effective font-size of your root (or html) element.
</p>
<p>
  Ideally you should not be setting the font-size on the html or root element in
  order to allow it to scale with user-preference, in which case this should be
  set to 16px.
</p>
<h5>Default value</h5>
<Code code={`$govie-root-font-size: 16px;`} />

<h4>$govie-typography-scale</h4>
<p>Responsive typography font map.</p>
<p>
  This is used to generate responsive typography that adapts according to the
  breakpoints.
</p>
<p>
  Font size and font weight can be defined for each breakpoint. You can define
  different behaviour on tablet and desktop. The 'null' breakpoint is for
  mobile.
</p>
<p>
  Line-heights will automatically be converted from pixel measurements into
  relative values. For example, with a font-size of 16px and a line-height of
  24px, the line-height will be converted to 1.5 before output.
</p>
<p>
  You can also specify a separate font size and line height for print media.
</p>

<h3>Internet Explorer 8</h3>

  included in the generated CSS once, no matter how many times they are imported

<h4>govie-typography-responsive</h4>
<p>Responsive typography helper.</p>
<p>
  Takes a point from the responsive 'font map' as an argument (the size as it
  would appear on tablet and above), and uses it to create font-size and
  line-height declarations for different breakpoints, and print.
</p>
<p>Example font map:</p>
<p>{`19: (
  null: (
    font-size: 16px,
    line-height: 20px
  ),
  tablet: (
    font-size: 19px,
    line-height: 25px
  ),
  print: (
    font-size: 14pt,
    line-height: 1.15
  )
);`}</p>

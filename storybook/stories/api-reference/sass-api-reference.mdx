# Sass API Reference

## Settings

### Assets

#### $govie-assets-path

Path to the assets directory, with trailing slash.

This is the directory where the images and fonts subdirectories live. You will need to make this directory available via your application - see the README for details.

##### Default value

```css
$govie-assets-path: '/assets/';
```

#### $govie-images-path

Path to the images folder, with trailing slash.

##### Default value

```css
$govie-images-path: '#{$govie - assets - path}images/';
```

#### $govie-fonts-path

Path to the fonts folder, with trailing slash.

##### Default value

```css
$govie-fonts-path: '#{$govie - assets - path}fonts/';
```

### Colours

#### $govie-brand-colour

Brand colour.

##### Default value

```css
$govie-brand-colour: govie-colour('blue');
```

#### $govie-text-colour

Text colour.

##### Default value

```css
$govie-text-colour: govie-colour('black');
```

#### $govie-body-background-colour

Body background colour.

##### Default value

```css
$govie-body-background-colour: govie-colour('white');
```

#### $govie-print-text-colour

Text colour for print media.
Use 'true black' to avoid printers using colour ink to print body text.

##### Default value

```css
$govie-print-text-colour: #000000;
```

#### $govie-secondary-text-colour

Secondary text colour.

Used in for example 'muted' text and help text.

##### Default value

```css
$govie-secondary-text-colour: govie-colour('dark-grey');
```

#### $govie-focus-colour

Focus colour.

Used for outline (and background, where appropriate) when interactive elements (links, form controls) have keyboard focus.

##### Default value

```css
$govie-focus-colour: govie-colour('yellow');
```

#### $govie-focus-text-colour

Focused text colour.

Ensure that the contrast between the text and background colour passes WCAG Level AA contrast requirements.

##### Default value

```css
$govie-focus-text-colour: govie-colour('black');
```

#### $govie-focus-text-colour

Focused text colour.

Ensure that the contrast between the text and background colour passes WCAG Level AA contrast requirements.

##### Default value

```css
$govie-focus-text-colour: govie-colour('black');
```

#### $govie-error-colour

Error colour.
Used to highlight error messages and form controls in an error state.

##### Default value

```css
$govie-error-colour: govie-colour('red');
```

#### $govie-success-colour

Success colour.
Used to highlight success messages and banners.

##### Default value

```css
$govie-success-colour: govie-colour('green');
```

#### $govie-border-colour

Border colour.
Used in for example borders, separators, rules and keylines.

##### Default value

```css
$govie-border-colour: govie-colour('mid-grey');
```

<h4>$govie-root-font-size</h4>
<p>Root font size.</p>
<p>
  This is used to calculate rem sizes for the typography, and should match the
  effective font-size of your root (or html) element.
</p>
<p>
  Ideally you should not be setting the font-size on the html or root element in
  order to allow it to scale with user-preference, in which case this should be
  set to 16px.
</p>
<h5>Default value</h5>

```css
$govie-root-font-size: 16px;
```

<h4>$govie-typography-scale</h4>
<p>Responsive typography font map.</p>
<p>
  This is used to generate responsive typography that adapts according to the
  breakpoints.
</p>
<p>
  Font size and font weight can be defined for each breakpoint. You can define
  different behaviour on tablet and desktop. The 'null' breakpoint is for
  mobile.
</p>
<p>
  Line-heights will automatically be converted from pixel measurements into
  relative values. For example, with a font-size of 16px and a line-height of
  24px, the line-height will be converted to 1.5 before output.
</p>
<p>
  You can also specify a separate font size and line height for print media.
</p>
<h5>Default value</h5>
```css $govie-typography-scale: ( 80: ( null: ( font-size: 53px, line-height: 55px
), tablet: ( font-size: 80px, line-height: 80px ), print: ( font-size: 53pt, line-height:
1.1 ) ), 48: ( null: ( font-size: 32px, line-height: 35px ), tablet: ( font-size:
48px, line-height: 50px ), print: ( font-size: 32pt, line-height: 1.15 ) ), 36: (
null: ( font-size: 24px, line-height: 25px ), tablet: ( font-size: 36px, line-height:
40px ), print: ( font-size: 24pt, line-height: 1.05 ) ), 27: ( null: ( font-size:
18px, line-height: 20px ), tablet: ( font-size: 27px, line-height: 30px ), print:
( font-size: 18pt, line-height: 1.15 ) ), 24: ( null: ( font-size: 18px, line-height:
20px ), tablet: ( font-size: 24px, line-height: 30px ), print: ( font-size: 18pt,
line-height: 1.15 ) ), 19: ( null: ( font-size: 16px, line-height: 20px ), tablet:
( font-size: 19px, line-height: 25px ), print: ( font-size: 14pt, line-height: 1.15
) ), 16: ( null: ( font-size: 14px, line-height: 16px ), tablet: ( font-size: 16px,
line-height: 20px ), print: ( font-size: 14pt, line-height: 1.2 ) ), 14: ( null:
( font-size: 12px, line-height: 15px ), tablet: ( font-size: 14px, line-height: 20px
), print: ( font-size: 12pt, line-height: 1.2 ) ) ); ```

<h4>$govie-grid-column</h4>
<p>Generate grid column styles.</p>
<p>Creates a grid column with standard gutter between the columns.</p>
<p>Grid widths are defined in the `$govie-grid-widths` map.</p>
<p>
  By default the column width changes from 100% to specified width at the
  'tablet' breakpoint, but other breakpoints can be specified using the $at
  parameter.
</p>
<h5>Parameters</h5>
<table>
  <tbody>
    <tr>
      <th>Name</th>
      <th>Description</th>
      <th>Type</th>
      <th>Default value</th>
    </tr>
    <tr>
      <th scope="row">`$width`</th>
      <td>name of a grid width from $govie-grid-widths</td>
      <td>`String`</td>
      <td>`full`</td>
    </tr>
    <tr>
      <th scope="row">`$float`</th>
      <td>left | right</td>
      <td>`String`</td>
      <td>`left`</td>
    </tr>
    <tr>
      <th scope="row">`$at`</th>
      <td>mobile | tablet | desktop | any custom breakpoint</td>
      <td>`String`</td>
      <td>`tablet`</td>
    </tr>
  </tbody>
</table>
<br />

<h5>Usage</h5>

```css
@include govie-grid-column($width: 'full', $float: 'left', $at: 'tablet');
```

<h5>Example</h5>

```css

<h3>Layout</h3>

<h4>govie-width-container</h4>
<p>Width container mixin</p>
<p>Used to create page width and custom width container classes.</p>
<h5>Parameters</h5>
<table>
  <tbody>
    <tr>
      <th>Name</th>
      <th>Description</th>
      <th>Type</th>
      <th>Default value</th>
    </tr>
    <tr>
      <th scope="row">`$width`</th>
      <td>Width in pixels</td>
      <td>`String`</td>
      <td>`$govie-page-width`</td>
    </tr>
  </tbody>
</table>
<h5>Usage</h5>
```css @include govie-width-container($width: '$govie-page-width'); ```

<h5>Example</h5>
```css
// Creating a 1200px wide container class
.app-width-container--wide {
  @include govie-width-container(1200px);
}
```

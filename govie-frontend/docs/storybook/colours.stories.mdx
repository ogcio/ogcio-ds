# Colours

- Colour palette ([\_colours-palette.scss](https://github.com/ogcio/ogcio-ds/blob/main/govie-frontend/package/govie/settings/_colours-palette.scss))
- Colour tokens ([\_colours-applied.scss](https://github.com/ogcio/ogcio-ds/blob/main/govie-frontend/package/govie/settings/_colours-applied.scss))

The colour tokens take the palette colours and apply them to specific scenarios, e.g. link colour, border colours etc.
